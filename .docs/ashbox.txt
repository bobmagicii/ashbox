ashbox.sh <command>...
======================

COMMANDS

	list <options>

		Show all certs being tracked by acme.sh.
		Formats: --csv, --tsv, --ssv, --json, Default: raw format.

	issue <...>

		Fetch SSL certs for the specified domains.
		Use without any arguments to see full usage and options.

	remove <...>

		Remove specified certs from the system.
		Use without any arguments to see full usage and options.

	update

		Download updates for both ashbox.sh and acme.sh.

	--version

		List ashbox and acme.sh versions.

SET DEFAULTS

	default:letsencrypt - use Let's Encrypt by default.
	default:zerossl - use ZerlSSL by default.

CONFIG TOOLS

	acme.sh Shell

	* conf:acmesh:shell:args
	  CLI args needed to make acme.sh work within the box.

	Ashbox Shell

	* conf:ashbox:shell:path
	  Shell script to export Ashbox onto PATH.

	Apache VHost SSL

	* conf:apache:vhost:ssl <domain:optional>
	  Without domain it will generate using $SSLDomain for VHost config args.

	Gitea SSL Configuration

	* conf:gitea <domain>
