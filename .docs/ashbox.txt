ashbox.sh <command>...
======================

COMMANDS

	list <options>

		Show all certs being tracked by acme.sh.
		Formats: --csv, --tsv, --ssv, --json, Default: raw format.

	issue <...>

		Fetch SSL certs for the specified domains. Use without any arguments to see full usage and options.

	remove <...>

		Remove specified certs from the system.

	update

		Download updates for both ashbox.sh and acme.sh.

	default:letsencrypt

		Set the default cert authority to Lets Encrypt.

	default:zerossl

		Set the default cert authority to ZeroSSL.

	conf:acmesh

		Print the command line arguments needed to make acme.sh work if choosing to call it directly.

	conf:apache <domain>

		Print the SSL config used by Apache using the $SSLDomain placeholder or a specified domain.
